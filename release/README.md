# Installing the Game
## To play on **Desktop**:
Java must be installed on your device.
* Click `Korona Kablam-i!.jar`
* Click "Download"
* Double click to run
   * Alternatively:
      * Open a terminal (cmd)
      * Navigate to the file `$cd [path]`
      * Run `$java -jar Suicide.jar`

## To play on Andoid:
* Click `Korona Kablam-i!.apk`
* Download the file.
* Disable any operating system warnings about installing the app
* Install and play
